* 補足
** GitHub補足
- masterブランチは常に最新にしよう
  - git checkout master
  - git fetch
  - git pull origin master

- 作業が完了してPull Reqする前に，masterをマージしておこう
  - git checkout my_branch
  - git merge master
  - （conflictの解消）
  - git stage .
  - git commit
  - git push origin my_branch
  - （pull requestの送信）
