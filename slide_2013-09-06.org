* RSpec入Railsの作り方
  - test_unitなしで作成

    rails new rails_rspec -T

  - Gemfileを編集

    group :development, :test do
      gem 'rspec-rails', '~> 2.0'
    end

  - Bundler

    bundle install
    rails generate rspec:install

  - db

    rake db:migrate

  - 参考

    https://github.com/rspec/rspec-rails
